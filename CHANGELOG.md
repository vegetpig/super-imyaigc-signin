# 更新日志

## v0.1.1 - 2026-07-04

### 修复

- 签到脚本现在会保存签到后的 `post-signin-*.png` 截图，方便人工复核。
- 签到成功判定改为优先识别页面明确文案 `今日已成功签到`，并辅助检查当天日历格子的已签到状态。
- 签到前会尝试关闭或推进新手引导遮罩，避免遮罩挡住签到入口。
- 避免旧逻辑只要页面存在任意已签到图标就误报成功。

## v0.1.0 - 2026-07-04

首个可安装版本。

### 已包含

- IMYAI 登录、Cookie 保存和登录刷新脚本。
- IMYAI 每日签到能力。
- 聊天模型调用：模型列表、模型搜索、官方 groupId、Codex 工作区 session。
- 图片生成：模型列表、自动选模型、参考图上传、任务轮询、结果下载。
- 网络策略：配置代理、直连、环境代理、本地 Clash/Mihomo 检测。
- 中文 README、安装文档、协作说明。
- Windows 快速安装脚本 `install.ps1`。
- 私有仓库多机同步所需的配置文件和 session 文件。
