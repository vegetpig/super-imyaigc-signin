# v0.1.0 初始可安装版本

这是 `super-imyaigc-signin` 的第一个正式发布版本，目标是让新电脑可以快速拉取、安装并开始使用 IMYAI Codex Skill。

## 快速安装

推荐使用 GitHub CLI：

```powershell
gh auth login
mkdir $env:USERPROFILE\.codex\skills -Force
cd $env:USERPROFILE\.codex\skills
gh repo clone vegetpig/super-imyaigc-signin
cd super-imyaigc-signin
powershell -ExecutionPolicy Bypass -File .\install.ps1
```

需要安装后直接验证：

```powershell
powershell -ExecutionPolicy Bypass -File .\install.ps1 -Verify -Phone YOUR_PHONE
```

也可以下载 release 附件 `super-imyaigc-signin-v0.1.0.zip`，解压后运行：

```powershell
powershell -ExecutionPolicy Bypass -File .\install.ps1
```

## 本版本包含

- IMYAI 登录、Cookie 保存和登录刷新。
- IMYAI 每日签到。
- 聊天模型调用、模型列表、模型搜索、官方 groupId、本地 session。
- 图片生成、自动选模型、参考图上传、任务轮询、结果下载。
- 网络策略：配置代理、直连、环境代理、本地 Clash/Mihomo 检测。
- 中文 README、快速安装文档、协作说明和变更日志。
- Windows 快速安装脚本 `install.ps1`。
