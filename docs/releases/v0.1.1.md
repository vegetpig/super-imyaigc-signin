# v0.1.1 签到验证修复

这个版本修复了签到脚本的成功判定问题。

## 修复内容

- 签到后新增 `post-signin-*.png` 截图，最终历史记录也会指向这张后置截图。
- 成功判定优先检查页面明确文案 `今日已成功签到`。
- 保留当天日历格子的已签到状态作为辅助确认。
- 签到前自动关闭或推进新手引导遮罩，避免遮罩阻挡点击。
- 旧的宽松 DOM 判断不再能单独决定最终成功状态。

## 验证结果

已在 2026-07-04 实测：

- 页面打开签到弹窗。
- 截图显示 `今日已成功签到`。
- 脚本最终返回 `OK`。

## 快速安装

```powershell
gh auth login
mkdir $env:USERPROFILE\.codex\skills -Force
cd $env:USERPROFILE\.codex\skills
gh repo clone vegetpig/super-imyaigc-signin
cd super-imyaigc-signin
powershell -ExecutionPolicy Bypass -File .\install.ps1
```

如果已经安装：

```powershell
cd $env:USERPROFILE\.codex\skills\super-imyaigc-signin
git pull --ff-only
```
