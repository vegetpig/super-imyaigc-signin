# v0.1.2 多账号签到支持

这个版本补全多账号签到配置，并修复不同账号首页状态下签到入口不一致的问题。

## 新增

- 新增账号 `THIRD_PHONE`。
- 确认账号 `SECOND_PHONE` 和 `THIRD_PHONE` 的密码已通过现有 `.secret_key` 机制加密保存。
- README 新增“多账号签到”说明。

## 修复

- 签到入口支持两种路径：
  - 头像菜单里的 `签到领积分`。
  - 顶部图标区的签到入口。
- 修复头像菜单 selector 写法，避免 Playwright selector 解析失败。

## 验证结果

已在 2026-07-04 实测：

```text
YOUR_PHONE: OK
SECOND_PHONE: OK
THIRD_PHONE: OK
```

## 全账号签到

```powershell
cd $env:USERPROFILE\.codex\skills\super-imyaigc-signin

```
