# v0.1.3 连续签到天数输出

这个版本新增连续签到天数输出。

## 新增

- 签到成功后从页面文案中提取 `已连续签到X天`。
- 日志输出格式：

```text
Consecutive sign-in days: X
```

## 实测结果

2026-07-04 实测：

```text
YOUR_PHONE: OK, Consecutive sign-in days: 2
SECOND_PHONE: OK, Consecutive sign-in days: 1

```

## 运行命令

cd $env:USERPROFILE\.codex\skills\super-imyaigc-signin
