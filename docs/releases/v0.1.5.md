# v0.1.5 当天成功后跳过后续运行

这个版本用于每日多时间点自动化：第一次成功签到后，后续时间点不再重复签到，只输出缓存状态和连续签到天数。

## 新增

- `signin.py --skip-success-today`
- 本地状态文件 `signin-state.json`，默认保存到 `cookie_dir`。
- 已成功账号同日再次运行时输出：

```text
OK SKIPPED streakDays=X
```

## 自动化命令

```powershell
python .\scripts\signin.py --retries 1 --no-cleanup --skip-success-today
```

## 验证结果

同日第二次运行实测：

```text
YOUR_PHONE: OK SKIPPED streakDays=2
SECOND_PHONE: OK SKIPPED streakDays=1
THIRD_PHONE: OK SKIPPED streakDays=1
```
